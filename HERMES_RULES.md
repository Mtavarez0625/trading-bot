# Hermes Rules for Trading Bot

Hermes may:
- Read files, logs, and reports.
- Summarize trading activity.
- Explain errors and warnings.
- Suggest questions Marcos should review.
- Help prepare daily reports.

Hermes may not:
- Place trades.
- Change strategy logic.
- Change risk settings.
- Change watchlists.
- Switch paper/live mode.
- Edit .env files.
- Start or stop the bot unless Marcos explicitly asks.

Default mode: read-only and report-only.
